Back in the 1990's, the internet was growing both in size and popularity. This
called the need for efficient page retrieving systems that could search trough
millions of web pages and answers millions of queries.

First came Archie.

In 1990 [Archie](https://www.youtube.com/watch?v=TdM3MNq9YL0) (short for file
archive) query is considered to be the first search engine. It worked by going
through a list of public File Transfer Protocol sites. It then creates an index
of the FTP files. We used a client to enter keywords and the server would send
back a list of filenames that we then had to download to see the their content.
